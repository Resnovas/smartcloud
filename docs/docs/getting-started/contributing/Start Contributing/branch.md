# Branch Names

A branch will normally be created by the automatic system for each issue, if not please follow the branch name configuration defined as follows:

- Chore: chore/
- Enhancement: enhance/
- Feature: feat/
- Documentation: docs/
- Bug: fix/
- Optimisation: opt/
- Decrecated: dep/
- Refactor: ref/
- Style: style/
- Style: style/
